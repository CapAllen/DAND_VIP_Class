# 数据分析入门VIP班导学及项目
## 说明

为方便[Udacity](https://cn.udacity.com/)数据分析入门VIP班学员和助教查阅及共享而建立此项目，之后会不断地进行迭代、更新和沉淀，践行Udacity**自律、共享、探索**的价值观。  

此项目主要包含以下内容：

- **每周导学**：以周为单位，设定学习内容与学习目标，引导学生按进度完成课程，总共十四周。
- **Uday-Notebook**：优达日所用Notebook。
- **Uday-Video**：优达日录屏视频。
